Demo & Integration Guide

Embeddable patient medication management system for your healthcare application

Quick Start

Get started in seconds by adding a single script tag to your HTML. The widget will automatically initialize and create a container.

<!-- Add this to your HTML -->
<script src="http://127.0.0.1:5500/embed/index.js" data-api-key="your-api-key-here" data-primary-color="#5fc9af"></script>
<!-- Or create a custom container -->
<div id="patient-widget"></div>
<script src="http://127.0.0.1:5500/embed/index.js" data-container-id="patient-widget" data-api-key="your-api-key"></script>

API Documentation

Script Attributes

Attribute Description Default
data-api-key Your API authentication key -
data-container-id ID of the container element patient-widget
data-primary-color Main theme color #57CC99
data-text-color Primary text color #212121
data-background-color Page background color #FBFBFB
data-card-background Card/container background #FFFFFF

JavaScript Methods

// Get widget instance
const widget = window.patientWidgetInstance;
// Navigate to different views
widget.navigateTo('dashboard');
widget.navigateTo('schedule');
widget.navigateTo('medicine');
widget.navigateTo('report');
widget.navigateTo('detail', 'patient1');
widget.navigateTo('add-medicine');
widget.navigateTo('add-prescription');
// Update theme dynamically
widget.setTheme({ primaryColor: '#ff6b6b', textColor: '#2d3436' });
// Reload widget
widget.reload();
// Destroy widget
widget.destroy();

Widget Events

Event Description Data
patient-widget:navigate Fired when navigating between views { view, patientId }
patient-widget:patient-selected Fired when a patient is selected { patientId }
patient-widget:medicine-saved Fired when medicine is saved { medicine data }
patient-widget:prescription-saved Fired when prescription is saved { prescription data }
patient-widget:schedule-deleted Fired when schedule is deleted { scheduleId }
patient-widget:medicine-deleted Fired when medicine is deleted { medicineId }
patient-widget:report-generated Fired when report is generated { filters }
// Listen to events
const container = document.getElementById('patient-widget');
container.addEventListener('patient-widget:navigate', function(e) { console.log('Navigation:', e.detail); });
container.addEventListener('patient-widget:medicine-saved', function(e) { console.log('Medicine saved:', e.detail);
// Sync with your backend });

Event Handling

const widget = document.getElementById('patient-widget');
// Track navigation
widget.addEventListener('patient-widget:navigate', (e) => { console.log('User navigated to:', e.detail.view); });
// Sync medicine data
widget.addEventListener('patient-widget:medicine-saved', async (e) => { await fetch('/api/medicines', { method: 'POST', body: JSON.stringify(e.detail) });
});

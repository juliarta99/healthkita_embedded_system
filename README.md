# HealthKita Embedded System

HealthKita adalah sistem manajemen layanan kesehatan yang dapat diintegrasikan ke dalam website rumah sakit, klinik, atau platform kesehatan lainnya. Sistem ini menyediakan berbagai fitur seperti manajemen obat, resep, jadwal dokter, detail pasien, dan laporan medis.

## Quick Start

### 1. Demo Embedded System
Folder ini berisi **contoh implementasi** bagaimana HealthKita System dapat diintegrasikan ke dalam website yang sudah ada.

**Akses Demo:**
- **Demo Implementasi:** [https://rs-jaya-giri.vercel.app/healthkita.html](https://rs-jaya-giri.vercel.app/healthkita.html)

Gunakan file-file di folder ini sebagai **referensi** untuk melihat bagaimana sistem HealthKita tampil ketika sudah diintegrasikan ke website Anda.

---

### 2. Embed System (Sistem Utama)
Ini adalah **core system** dari HealthKita yang dapat di-embed ke website manapun. Semua fitur utama ada di sini.

- **Akses Sistem:** [https://healthkita-embedded-system.vercel.app/](https://healthkita-embedded-system.vercel.app/)

**File Utama:**
- `index.js` - Logic utama sistem, handling interaksi, data management, dan komunikasi antar modul

---

### 3. Integration Guide
Folder ini berisi **panduan lengkap** bagaimana cara mengintegrasikan HealthKita System ke website Anda.

**Akses Dokumentasi:**
- **Panduan Integrasi:** [https://guide-embed-system-healthkita.vercel.app/](https://guide-embed-system-healthkita.vercel.app/)
